PythonQt is hosted on SourceForge.

You can download the source code as a tarball at http://sourceforge.net/projects/pythonqt/files/. Alternatively you can get the latest version from the svn repository.

You can also browse the source code online via ViewVC: http://pythonqt.svn.sourceforge.net/viewvc/pythonqt/trunk/

Note
We do not offer prebuilt binaries, since there are so many possible combinations of platforms (Windows/Linux/MacOs), architectures (32/64 bit) and Qt / Python versions.
